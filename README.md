# 💱 Conversor de Monedas - Java

Este es un proyecto desarrollado como parte del desafío del programa Oracle Next Education (ONE), en el cual se construyó un **Conversor de Monedas** utilizando Java y una API real de tasas de cambio.

---

## 🚀 Descripción del proyecto

El Conversor de Monedas es una aplicación de consola que permite al usuario:

- Consultar las tasas de cambio actualizadas desde una API.
- Ingresar un monto en dólares (USD).
- Convertirlo a distintas monedas de América Latina.
- Visualizar el resultado de la conversión de forma clara y sencilla.

---

## 🌍 Monedas disponibles

El usuario puede convertir dólares estadounidenses (USD) a las siguientes monedas:

- ARS – Peso argentino 🇦🇷  
- BOB – Boliviano 🇧🇴  
- BRL – Real brasileño 🇧🇷  
- CLP – Peso chileno 🇨🇱  
- COP – Peso colombiano 🇨🇴  
- USD – Dólar estadounidense 🇺🇸

## 🛠 Tecnologías y herramientas

- **Java 11+**
- **HttpClient (Java estándar)**
- **Gson (Google)** para procesar JSON
- **ExchangeRate-API**: API pública para tasas de cambio
- **IntelliJ IDEA** como entorno de desarrollo
- **Scanner** para entrada de datos por consola

---

## 🔧 Estructura del proyecto

- `ConversorApp.java`: Clase principal con el menú y lógica completa.
- Uso modular de métodos para mostrar menú, obtener tasas y hacer conversiones.
- Lógica organizada en bucles para interacción continua hasta que el usuario decida salir.

---

## 📥 Instalación y ejecución

### Requisitos:

- Java 11 o superior
- Conexión a internet
- Biblioteca `gson-<version>.jar` agregada al proyecto

### Pasos:

1. Clona o descarga este repositorio.
2. Abre el proyecto en IntelliJ o tu IDE favorito.
3. Asegúrate de agregar la biblioteca Gson a tu proyecto.
4. Ejecuta el archivo `ConversorApp.java`.

---

## 🧠 Aprendizajes clave

- Uso de `HttpClient`, `HttpRequest` y `HttpResponse` en Java moderno.
- Lectura y manipulación de JSON con la biblioteca Gson.
- Diseño de un programa interactivo con entrada por consola.
- Práctica de buenas prácticas como modularidad, validación y separación de responsabilidades.
